PROJECT TITLE:Car Database System
VERSION or DATE: 06/10/2017
HOW TO START THIS PROJECT:
1.Start the syatem by selecting 'start'.
2.You can search the cars by registration number, car make, year made, colour and car age.
3.You can add a car to the database.
4.You can delete a car from the database.
5.You can edit the car colour and car price.
